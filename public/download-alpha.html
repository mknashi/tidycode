System Requirements

macOS

  • macOS 10.15 (Catalina) or later
  • Apple Silicon (M1/M2/M3) or Intel processor
  • 50 MB free disk space

Windows

  • Windows 10 (version 1809 or later) for x64 builds
  • Windows 11 for ARM64 builds
  • 50 MB free disk space

Installation Notes

macOS: After downloading, open the .dmg file and drag Tidy Code to your Applications folder. On first launch, you may need to right-click the app and select "Open" to bypass Gatekeeper.

Windows: Run the installer and follow the prompts. For ARM64 devices (like Surface Pro X), use the ARM64 build. Windows Defender may show a warning for unrecognized apps - click "More info" and then "Run anyway" to proceed.
