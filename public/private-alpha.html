What Is Tidy Code?

Tidy Code is a focused workspace for the everyday developer tasks that happen outside your main IDE. You know the drill: quickly viewing a PDF spec, formatting messy JSON from an API response, cleaning up CSV data, or just jotting down notes while debugging.

Instead of opening five different apps or websites, Tidy Code handles it all in one clean interface. No bloat, no distractions â€” just the tools you actually need, available on desktop and web.

What's Included in the Alpha

Multi-Tab Editing

  • Open multiple files with drag-and-drop tab reordering
  • Syntax highlighting for 50+ languages
  • Optional VIM mode for keyboard-driven editing
  • Dark and light themes

Document Viewers

  • PDF viewer with search, zoom, navigation, and distraction-free focus mode
  • SVG viewer with pan and zoom controls
  • Markdown live preview with side-by-side editing

Formatters & Data Tools

  • JSON/XML/YAML formatting with syntax validation and structure view
  • CSV editing with preview and column management
  • Error detection and helpful validation messages
  • Auto-pairing for brackets, quotes, and XML tags

Developer Productivity

  • Advanced search with regex support and adjustable results panel
  • Large file support (100MB+) with WASM acceleration â€” 8x faster indexing, 14x faster search
  • File Explorer for navigating project structures
  • Built-in notes and todo lists for tracking tasks
  • LSP support for code intelligence
  • AI-powered coding assistance (experimental)

Desktop & Web

  • Native desktop apps for Windows and macOS
  • Web version with same core features
  • Offline-first behavior on desktop
  • All data stays local by default
  • No ads, no tracking
